# intdiv関数
PHP7以降で使える、小数点以下を切り捨てて整数を返す関数。負の数にも対応。
```php
<?php
$result = intdiv(10, 3);
echo $result;  // 出力: 3
?>
```
  
# (int)キャスト
小数点以下を切り捨てる。PHP7以前の古いバージョンでも使える。
```php
<?php
$result = (int)(10 / 3);
echo $result;  // 出力: 3
?>
```
  
# その他の関数
|関数|説明|例|
|---|---|---|
|floor()|小数点以下を切り捨て|floor(10 / 3) → 3|
|ceil()|小数点以下を切り上げ|ceil(10 / 3) → 4|
|round()|四捨五入|round(10 / 3) → 3|
