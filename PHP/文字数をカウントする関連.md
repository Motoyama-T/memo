# 重複を許してカウントするには
**■substr_count関数は重複をカウントしない**  
```php
$text = "aaa";
echo substr_count($text, "aa");  // 出力: 1
```
"aaa"の中には"aa"が2回重なって見えますが、substr_count関数は最初の"aa"を見つけた時点で次の検索を"a"の次から始めるため、重複部分はカウントされない。
